# mdtracker
 Native music tracker for SEGA MEGA DRIVE / GENESIS / NOMAD
 
 Works on:
 
	BlastEm (RetroArch)
	
	Mega Everdrive Pro
	
 Partially:
 
	PicoDrive (RetroArch)
	
	- No CSM
	- No DAC panning
	- Wrong PSG noise pitch

Code::Blocks 20.03

Change path in compiler settings to where SGDK is.

/inc - copied SGDK headers.

/res - graphics, samples etc.
